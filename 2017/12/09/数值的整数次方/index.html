数值的整数次方

题目描述

给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。

思路

快速幂（要认识到任何一个整数都能用二进制表示，其中内涵很深）
若要求 3^20 ，按一般的方法，要进行 19 次乘法才能得到结果，但是可以注意到 3^20 = 9^10 = 81^5 ，那么只要 4 次运算就能得到结果。
现在求 A 的 156 次方，将 156 化为二进制 156（10）= 10011100（2），发现 A^156 = (A^4 ) (A^8 ) (A^16 ) (A^128 )。从二进制的右端算到左端，用位移符 >> 计算。

知识点

  • 快速幂取模运算
  • 矩阵快速幂
